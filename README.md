# fishsinker

# What is it?
Fishsinker is a TUI with built in automation scripts for game X.
It targets various different sub-games such as:
1. Fishing
    - Barn
    - Instant
2. X
3. X
4. X
5. X
6. X

# How to use it?
- Upon running the app, you will be presented with a simple UI.
- You may exit, or proceed to the macros.
    - Upon proceeding, you can decide which macro to run,
        - Instruction are provided on the keyboard shortcuts to use.
